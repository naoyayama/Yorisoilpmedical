# 医療機関向けAIアシスタント「よりそい」ランディングページ

このプロジェクトは、医療機関の診察業務を効率化するAIアシスタント「よりそい」の紹介用ランディングページ（LP）です。

## 🚀 プロジェクト概要
医師と患者の自然な会話から、電子カルテに必要な「S.O.A.P.形式」の記録を自動生成するSaaS製品の魅力を伝えるためのウェブページです。

## ✨ 特徴
- **モダンでプレミアムなデザイン**: 医療機関にふさわしい清潔感と、最新テクノロジーを感じさせる洗練されたUI。
- **動的なユーザー体験**: AOS.jsライブラリを使用したスクロール連動アニメーション、ヒーローセクションのタイピング・波形アニメーション。
- **B2B特化型フォーム**: 問い合わせや資料請求がスムーズに行える、入力しやすいフォーム設計。
- **レスポンシブ対応**: PC、タブレット、スマートフォンのすべての端末で最適に表示されます。
- **セキュリティの可視化**: 厚生労働省・経済産業省・総務省のガイドラインに準拠した堅牢なシステム構成を視覚的に表現。

## 🛠 使用技術
- **HTML5 / CSS3**
- **Tailwind CSS** (CDN)
- **AOS.js** (Animate On Scroll)
- **Google Fonts** (Inter, Noto Sans JP)

## 📁 フォルダ構成
- `index.html`: LPのメインファイル（構造とスタイル）
- `medical_trust.png`: セキュリティ信頼性を表すイラスト
- `medical_trust_illustration.png`: 検討用のアセット（元画像）
- `yorisoi_lp_export.zip`: プロジェクト一式をまとめたアーカイブ

## 📦 プレビュー方法
1. このリポジトリをクローンまたはダウンロードします。
2. フォルダ内の `index.html` を任意のブラウザで開きます。
