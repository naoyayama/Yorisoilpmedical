# 医療機関向けAIアシスタント「よりそい」ランディングページ

このリポジトリは、医療機関の診察業務を効率化するAIアシスタント「よりそい」の紹介用ランディングページ（LP）のソースコードを一式含んでいます。

## 🚀 プロジェクトの特徴（再現性について）
このプロジェクトは、**「エンジニアがクローンして即座に、かつ完全な形で再現できること」**を最重要視して構成されています。

- **ポータビリティ**: 外部画像ファイルへの依存を排除し、アイコン（SVG）やスタイル（Tailwind CSS）をすべてコード内に内包、または信頼性の高いCDN経由で読み込んでいます。
- **完全自己完結**: `index.html` ファイル一つをブラウザで開くだけで、アニメーションやレスポンシブデザインを含むすべての機能が100%のクオリティで再現されます。

## ✨ 主な機能
- **動的アニメーション**: AOS.jsによるスクロール連動、SOAP入力シミュレーション、音声波形アニメーション。
- **モダンなUI構成**: グラスモーフィズムを採用したプレミアムなデザインと、清潔感のある配色。
- **堅牢なセキュリティ表現**: 最新のセキュリティ基準（3省2ガイドライン等）を視覚化したカスタムコンポーネント。

## 📁 構成ファイル
- `index.html`: LPの本体。HTML、CSS、JSがこの1ファイルに集約されています。
- `yorisoi_lp_export.zip`: 開発初期の素材やHTMLを含めた、オフライン配布用のバックアップ。
- `.gitignore`: 不要なシステムファイルの混入を防ぐ設定。

## 📦 実行・確認方法
1. このリポジトリをクローンします。
   ```bash
   git clone https://github.com/naoyayama/Yorisoilpmedical.git
   ```
2. フォルダ内の `index.html` を任意のブラウザで開きます。
   - ※画像サーバーなどは不要で、ローカルでそのまま閲覧可能です。
